# Node.js Backend API 🔐

A REST API built with Node.js and Express that provides user authentication and notes management functionality. It supports secure authentication, CRUD operations for notes, and user profile management.

---

## 🚀 Features

- User registration and login
- JWT authentication
- Password reset via email
- Token refresh and logout
- CRUD operations for notes
- Filtering and search for notes
- User profile update (including avatar upload)

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT (JSON Web Token)
- bcrypt
- Mongoose
- Nodemailer
- Multer
- CORS
